constants.js — общие константы: пути ассетов колод, конфиг DECKS, коэффициенты размеров (RATIO, TOPBAR_OFFSET).
utils/array.js — утилиты массивов (сейчас только shuffle).
utils/assetPaths.js — генерация путей к изображениям карт: кандидаты для лицевой/рубашки с webp→png→jpg фолбэком.
hooks/useDeckDims.js — расчёт размеров колоды снизу и нижнего отступа стола, реакция на ресайз/ориентацию.
layout.js — вся логика раскладок: рабочая область computePlayRect, вычисление размера карты fitCardSize, позиции слотов makeLayout для всех раскладов.
components/ImageWithFallback.jsx — <img> с автоматическим переключением источников (webp→png→jpg) и скрытием при финальной ошибке.
components/FlyingCard.jsx — анимированная «летящая» карта от колоды к слоту (позиционирование, масштаб, поворот, колбэк по завершении).
components/TopBar.jsx — верхняя панель с кнопками управления колодой и отображением текущего расклада.
components/ZoomFlight.jsx — анимация увеличения карты из стопки в центр экрана (для просмотра).
components/ZoomFlightBack.jsx — анимация уменьшения карты из центра экрана обратно в слот.
components/CardZoomOverlay.jsx — оверлей для просмотра карты в увеличенном виде (использует ZoomFlight).
VirtualDeckPage.jsx — страница-оркестратор: состояние колоды/расклада, обработчики (взять/перевернуть/перемешать/отменить/очистить), расчёт слотов, рендер стола, верхняя панель и стопка карт.
cards.json — данные карт (имена, значения, слаги и пр.).
